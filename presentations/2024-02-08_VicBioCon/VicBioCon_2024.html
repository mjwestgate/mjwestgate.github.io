Building an archive

df <- read_csv("my_data.csv")
metadata <- read_md("my_metadata.qmd")

archive <- dwca() |>
  add_occurrences(df) |>
  add_metadata(metadata)

then:

  • build() - construct a DwCA zip file
  • check() - run tests to check for DwC compliance
  • publish() - send your data to the ALA publication API
